# Worth Find Pro (禄马真经)

MVP built with Next.js + Tailwind.

## Features

- Landing page (天机阁)
- Four-step calculator flow (四柱排盘)
- K-score calculation with factor breakdown
- Result page with radar chart
- Local talisman image export (PNG)
- Local persistence via browser storage
- Reserved anonymous report API contract

## Tech

- Next.js App Router + TypeScript
- Tailwind CSS
- Zustand + React Hook Form + Zod
- ECharts
- html-to-image

## Run

```bash
npm install
npm run dev
```

## Test

```bash
npm test
```

## Privacy

All personal inputs are calculated locally in browser storage. No phone/name/exact salary is uploaded by default.
